## 什么是MCP？

MCP（模型上下文协议）是一种开放协议，通过标准化的服务器实现，使AI模型能够安全地与本地和远程资源交互。这使AI助手能够通过访问文件、数据库、API和其他上下文服务来扩展其功能。
